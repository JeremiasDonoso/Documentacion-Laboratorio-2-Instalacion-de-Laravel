## 📖 Introducción

Laravel utiliza el patrón de arquitectura MVC (Modelo-Vista-Controlador), el cual permite organizar el código de manera estructurada y mantenible.

- **Modelos:** Representan la estructura de la base de datos y permiten interactuar con ella.
- **Vistas:** Son la interfaz gráfica que el usuario visualiza en el navegador.
- **Controladores:** Gestionan la lógica de la aplicación y conectan los modelos con las vistas.
- **Rutas:** Definen las URLs de la aplicación y determinan qué controlador o vista se ejecuta.

Esta arquitectura facilita el desarrollo, mantenimiento y escalabilidad de las aplicaciones web.
