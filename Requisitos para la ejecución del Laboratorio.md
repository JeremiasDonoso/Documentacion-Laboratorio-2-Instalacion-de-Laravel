▪ PHP versión 8.0 o superior.
▪ Composer última versión estable.
▪ Laravel Installer o crear proyecto con laravel new / composer create-project.
▪ Paquete de servidor web local. (Entorno de Desarrollo)(ej. XAMPP, WampServer o Laragon).
▪ Servidor web: Apache o Nginx
▪ Base de datos MySQL/MariaDB funcionando.
▪ Editor de código (Visual Studio Code recomendado).
▪ NPN: si fue necesario. Dejarlo en blanco de lo Contrario.
▪ Sistema Operativo: Windows/MacOS/Linux

En mi caso esto es lo que he utilizado para el laboratorio:
▪ PHP versión 8.2.12
▪ Composer versión 2.9.5
▪ Instalación de Laravel utilizando el comando en Windows PowerShell "composer global require laravel/installer".
▪ Servidor Web Apache.
▪ Base de Datos MySQL.
▪ Editor de Código Visual Studio Code versión 1.115.0
▪ Sistema Operativo Windows 11
